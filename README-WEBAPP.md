# IdeaSpark — Full Open Web App

هذه نسخة ويب/PWA كاملة ومفتوحة من IdeaSpark.

## طريقة الاستخدام
- ارفع محتويات هذا المجلد كما هي على أي استضافة ويب ثابتة مثل Netlify أو Vercel أو Cloudflare Pages أو استضافة cPanel.
- ملف التشغيل الرئيسي هو: `index.html`.
- يجب بقاء الملفات التالية بجانب `index.html`: `manifest.json`, `sw.js`, مجلد `vendor`, والأيقونات.

## ملاحظات مهمة
- النسخة مفعّلة كـ Full/Open عبر `data-build="full"`.
- تعمل كـ Web App / PWA ويمكن إضافتها للشاشة الرئيسية من المتصفح.
- بعض المزايا التي تعتمد على صلاحيات المتصفح مثل الميكروفون أو مشاركة النظام تحتاج HTTPS عند رفعها على الاستضافة.
- الذكاء الاصطناعي يحتاج إعداد مفتاح/Proxy حسب الإعدادات الموجودة داخل التطبيق.

## تشغيل محلي للتجربة
افتح المجلد بسيرفر محلي بسيط، ثم ادخل من المتصفح على العنوان المحلي. فتح الملف مباشرة قد يعطل بعض خصائص PWA بسبب قيود المتصفح.
